Ver$

`ver$(0)` returns the revision number of the SAFE X3 engine.  
The type of result is [Char](4gl_char.html).

# Example

```
  Local Char VERSION(100)
  VERSION=ver$(0)
```

# Comment

`ver$(1)` is deprecated and returns an empty string.
