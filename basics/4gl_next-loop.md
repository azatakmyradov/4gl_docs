Next loop

Use `Next` in [For](4gl_for.html) loops to end the loop.

# Syntax

```
Next VAR_LOOP
Next
```

* `VAR_LOOP` is the variable used in [For](4gl_for.html) `VAR_LOOP`=... syntaxes.
