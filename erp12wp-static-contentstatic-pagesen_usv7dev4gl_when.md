When

`When` is the keyword that indicates a branch in a [Case](4gl_case.html) alternative. The syntax is the following:

```
  Case EXPRESSION
    When EXPRESSION_LIST
       Statements...
    When EXPRESSION_LIST
       Statements...
    ...
    When Default
       Statements...
  Endcase
```

  
For more information, see the [Case](4gl_case.html) documentation.
