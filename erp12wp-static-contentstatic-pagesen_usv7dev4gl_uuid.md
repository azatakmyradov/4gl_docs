Uuid$

`uuid$` generates a new canconical (string) UUID.

# Syntax

```
STRING_UUID=uuid$
```

# Example

```
# Assign a UUID if not assigned yet
Local Char STRING_UUID(36)
STRING_UUID = uuid$
```
