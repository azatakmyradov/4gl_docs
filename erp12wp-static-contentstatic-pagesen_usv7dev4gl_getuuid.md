Getuuid

`getUuid` generates a new UUID.

# Syntax

```
MY_UUID=getUuid
```

# Example

```
# Assign a UUID if not assigned yet
Local Uuident MY_UUID
If MY_UUID=Nulluuid : MY_UUID=getUuid : Endif
```
